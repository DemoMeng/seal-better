生成电子印章，包含个人章和企业章



####
    
    com.mqz.seal.v1.prs: 个人章生成入口
        1.测试函数ZMain
        2.可生成base64或保存到本地
        3.本地文件：temp/private    
        
    com.mqz.seal.v1.cs: 企业章生成入口
        1.测试函数ZMain
        2.生成base64
        3.本地文件：temp/public
